# PY4B

**What is PY4B?**, PY4B (aKa Python For Beginners) is an open source tool, made to help beginners struggling in using Python/EXE Files.

**PY4B Has these Built In Tools:**

- Missing Package Installer
- Guide to Installing Python
- Smart File Launcher


> [!TIP]
> PY4B Freezing when Installing Packages are __Normal!__
